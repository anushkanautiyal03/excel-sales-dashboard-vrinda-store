# Vrinda Store Sales Analysis 

## Project Overview
Performed data cleaning, processing, and analysis on Vrinda Store's
sales data to build an annual sales dashboard in Excel.

## Dataset
- Source: Vrinda Store internal sales data
- Records: ~31,000 orders
- Columns: Order ID, Date, Category, Channel, Amount, State, Gender, Age, Status

## Steps Performed
1. Data Cleaning
   - Removed duplicates
   - Fixed null/blank values
   - Standardized Gender column (M/W → Men/Women)
   - Corrected data types (dates, numbers)

2. Data Processing
   - Created Age Group column (Adult/Senior/Teenager)
   - Extracted Month from Date column
   - Calculated order status categories

3. Data Analysis
   - Sales by Month
   - Sales by Gender
   - Top 5 States by Revenue
   - Sales by Channel
   - Age & Gender comparison
   - Order Status breakdown

## Dashboard Features
- Interactive slicers (Month, Category, Channel)
- KPI cards (Revenue, Orders, Delivery Rate)
- Charts: Bar, Donut, Pie, Column

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
